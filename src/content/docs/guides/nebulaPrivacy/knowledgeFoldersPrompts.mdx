<Aside type="danger" title="Warning">THIS INFORMATION STAYS ON THE SERVER! DO NOT UPLOAD SENSITIVE INFORMATION! WE DO NOT TREAT THIS DATA AS SENSITIVE!</Aside>

Using the **Knowledge Bases** and **Folders** functionalities in Nebula requires you to upload files to the server. These files are saved and backed up. As such, DO NOT upload sensitive information here.
**System Prompts** that are set [globally](../modifyingAdvancedParams#global-modification) and [per model](../modifyingAdvancedParams#per-model-modification) are saved to the server. DO NOT upload sensitive information here.
This information is saved to the server because it is meant to be reused. We chose to keep this information to avoid forcing you to recreate it every time you log in.
