Nebula is based on the [Open WebUI](https://github.com/open-webui/open-webui) interface and uses Ollama as the backend for running the large language models. All services that are part of Nebula are running fully localized on the VU Campus on our dedicated servers. This allows us full control over what data is saved and how the data is used.


All your communications with Nebula are encrypted via TLS, the standard protocol for web
encryption (read more [here](https://www.cloudflare.com/learning/ssl/transport-layer-security-tls/)).
